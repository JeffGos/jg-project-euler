# jg-project-euler
Solutions to Project Euler problems https://projecteuler.net/

Install with Makefile

make

CMAKE Installation

- install cmake
- install cmaketools
- install MS c/c++

- create sources.cmake
- include source files

In CMakeLists.txt
- include main.cc 
- include ${SOURCES}

Running with VS Code
- bring up command pallette (Cmd-Shift-P)
- CMake-Config
- CMake-Build
- CMake-Debug
